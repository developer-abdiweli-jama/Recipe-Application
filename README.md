## 🎯 Features

### 💎 Design & Aesthetics
* **Glassmorphism UI**: Translucent panels with elegant backdrop blurs.
* **Premium Animations**: Staggered fade-ins, image zooms, and smooth transitions.
* **Dark Mode**: A refined midnight-black theme with high-contrast accents.

### 🍽️ Culinary Exploration
* **Real-time Search**: Instant culinary matching with a glowing interactive search bar.
* **Immersive Details**: Magazine-style layouts with drop-cap typography and scroll progress tracking.
* **Skeleton Loaders**: High-end shimmering placeholders for a seamless loading experience.
* **Masterclass Integration**: Direct links to YouTube video procedures.

### 📱 Performance & Tech
* **Responsive Grid**: Fluid layout from mobile to wide-screen 4K displays.
* **API Driven**: Dynamic content fetched from TheMealDB.
* **Utility-First styling**: Clean, maintainable code using Tailwind CSS v4.

## ⚙️ Installation & Setup

```bash
# Clone project
git clone https://github.com/developer-abdiweli-jama/Recipe-Application.git
cd Recipe-Application
```

### Setup

```bash
npm install
```

### Running the App

```bash
npm run dev
```

Access at `http://localhost:5173`.
